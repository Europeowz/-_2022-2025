# 天津市中考录取分数对比 (2022—2025)

> 海河之畔，学府争辉。天津 16 个区 100+ 所高中四年录取分数线的可视化查询工具。

---

## 预览

纯单文件 HTML 应用，浏览器直接打开即可使用。深蓝 + 古铜金配色，杂志级排版，响应式设计。

---

## 功能

| 功能 | 说明 |
|------|------|
| 学校搜索 | 支持中文校名模糊搜索，输入即过滤 |
| 区域筛选 | 16 个行政区（市内六区 + 市九所 + 环城四区 + 远郊五区 + 滨海新区） |
| 类别筛选 | 市教委直属 / 区属市重点 / 区级重点 / 民办 |
| 数据范围筛选 | 全部年份 / 2023 年起 / 2022 年起 / 四年均有 |
| 多列排序 | 点击任意年份列头，按该年录取分数升序/降序排列 |
| 涨跌标注 | 年份间变化量（↑↓→），年度整体趋势 |
| 统计概览 | 收录学校总数、最高分、最低分，实时动态计算 |

## 数据来源

所有录取分数线数据均来源于：

> **天津市教育招生考试院 · 招考资讯网**  
> 官方网站：`www.zhaokao.net`

具体来源说明：

- **原始数据发布**：每年 7 月中下旬，天津市教育招生考试院通过招考资讯网发布《天津市高级中等学校招生录取工作综述》及各批次录取最低分
- **数据整理**：作者从招考资讯网历年公告中逐条提取各校录取分数线，手动录入为结构化数据
- **数据标识**：标注为 `null` 的字段表示该年份该校无公开数据或未招生

> ⚠️ **重要声明**：本工具为民间整理，仅供择校参考。录取分数线受当年试题难度、招生计划、报考人数等多重因素影响，存在年度波动。**请以天津市教育招生考试院官方发布信息为准。**

## 技术实现

| 维度 | 技术选择 |
|------|----------|
| 语言 | HTML + CSS + JavaScript |
| 依赖 | **零外部依赖**，单文件纯手写 |
| 数据 | JavaScript 内联结构化数组，111 条学校记录 |
| CSS | 自定义设计系统（CSS 变量驱动），无框架 |
| 字体 | Noto Serif SC（衬体标题）+ DM Sans（正文）+ DM Mono（数据） |
| 响应式 | 适配桌面端（1360px）到移动端（375px） |

### 设计系统

```
主色调：   深藏蓝 #0a2540（导航、表头、标题）
强调色：   古铜金 #c9953c（装饰线、图标、交互高亮）
底色：     暖奶油 #f8f4ef（页面背景）
卡片：     纯白 #ffffff + 微阴影
区域标签： 16 区各自独立配色（CSS 变量按区分配）
分割线：   暖灰 #efe8df
```

### 关键实现细节

- **区域标签配色系统**：每个行政区有独立的 CSS 配色变量（`--badge-bg` / `--badge-text` / `--badge-border`），通过 `[data-district="..."]` 属性选择器自动匹配
- **筛选逻辑**：搜索、区域、类别、数据范围四重条件叠加（AND 逻辑），实时渲染
- **排序算法**：`null` 值始终排末尾，非空值按 `dir`（1/-1）升序降序
- **趋势计算**：取第一个有数据年份与最后一个有数据年份比较，差值决定 ↑↓→
- **SVG 波浪分割线**：纯 CSS/SVG 实现，无图片依赖
