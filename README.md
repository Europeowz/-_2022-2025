# 天津中考录取分数对比 (2022-2025)

天津 16 个区 100+ 所高中四年录取分数线可视化对比工具。

## 功能

- 收录 100+ 所学校 2022-2025 年录取分数线
- 按区域筛选（市内六区 + 市九所 + 环城 + 远郊）
- 按学校类别筛选（市教委直属 / 区属市重点 / 区级重点 / 民办）
- 按数据完整性筛选（仅显示有多年数据的学校）
- 多列排序（点击年份列头按分数排序）
- 涨跌趋势标注（↑↓→）
- 年份间变化量显示

## 技术

- **纯单文件 HTML**，零依赖，零框架
- 自定义设计系统（CSS 变量驱动）
- 深蓝 + 金色配色，Noto Serif SC 衬体标题
- 响应式布局，移动端适配
- 数据来源：天津市教育招生考试院 · 招考资讯网

## 预览

直接用浏览器打开 `天津中考录取分数对比_2022-2025.html` 即可。

## 开发说明

独立完成的全栈单文件应用：手写 100+ 所学校数据、CSS 设计系统、JavaScript 交互逻辑（搜索、筛选、排序、统计）。
